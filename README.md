# 🚀 Ejercicios Prácticos: FastAPI y Backend con Python

Este repositorio contiene una serie de **10 ejercicios prácticos progresivos** diseñados para dominar el desarrollo backend utilizando **FastAPI**, **Pydantic** y lógica de servidores, siguiendo el mapa de aprendizaje del ecosistema de Python.

## 🛠️ Tecnologías Utilizadas
* **Python** (Programación orientada a objetos y lógica backend)
* **FastAPI** (Creación de APIs REST rápidas y eficientes)
* **Pydantic** (Validación estricta de estructuras de datos)

## 📂 Contenido del Repositorio
Cada archivo `.py` resuelve un concepto clave del desarrollo backend de forma aislada:
1. `ejercicio_1_hola_api.py` - Fundamentos y Rutas Básicas (`GET`)
2. `ejercicio_2_query_params.py` - Parámetros de Búsqueda y Filtrado
3. `ejercicio_3_pydantic_post.py` - Validación de datos en el cuerpo (`POST`)
4. *(¡Y acá vas sumando los nombres de los que vayas completando!)*
